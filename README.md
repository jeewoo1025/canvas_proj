# HTML Canvas를 사용하여 슈퍼펭귄 앱 화면 묘사하기

슈퍼펭귄에 대해 간략히 설명하자면 2012년 영국에서 출시한 러닝 액션게임으로 간단한 터치를 통해 장애물을 피하며 아이템을 획득하는 게임이다.
Canvas를 사용하여 슈퍼펭귄의 3가지 화면을 묘사하였다.
